1.2 配置ssh key

由于使用hexo提交代码到github需要你git账号用户名密码，直接使用的话会很不安全，因此使用ssh key 解决客户端到服务器的连接问题

如果提示：No such file or directory 说明这是第一次使用git

此时ssh密钥生成完毕,打开github,点击settings,新建一个ssh,名字任意,然后输入刚才那一坨密钥。

添加好以后在回到本地bash输入

如果出现 Hi,你的名字! 那就是ok了
